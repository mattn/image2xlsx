# image2xlsx

Convert image file to Microsoft Excel file.

![image2xlsx](https://gyazo.compile-error.net/7244410693266b39.png)

## Usage

```
$ image2xlsx /path/to/image.jpg
```

This generate `/path/to/image.xlsx`.

## Installation

```
go get github.com/mattn/image2xlsx
```
